# Cobros V12 Campo

Proyecto limpio para construir APK de Cobros V12 Campo.

## Contenido

- `main.py`: aplicación principal.
- `buildozer.spec`: configuración de Buildozer.
- `.github/workflows/build-apk.yml`: compilación automática en GitHub Actions.
- `assets/icon.png`: ícono básico de la app.

## Importante

No subir bases locales `.db`, `.sqlite` ni carpetas `bin/` o `.buildozer/`.
